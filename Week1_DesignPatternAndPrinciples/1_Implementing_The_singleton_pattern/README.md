# Singleton Pattern - Java

## Objective
Implement the Singleton Design Pattern in Java.

## Folder Structure
```
1_Implementing_The_singleton_pattern
├── src
│   ├── Logger.java
│   └── Main.java
├── output
│   └── logger_output.png
└── README.md
```

## Description
The Singleton Pattern ensures that only one instance of the Logger class is created throughout the application.

## Files
- Logger.java – Implements the Singleton class.
- Main.java – Tests the Singleton implementation.
- logger_output.png – Screenshot of the program output.

## Output
The program demonstrates that both objects refer to the same Logger instance.
