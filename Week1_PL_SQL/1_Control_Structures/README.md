# PL/SQL Control Structures

## Objective

Implement PL/SQL control structures using Cursors, FOR LOOP, IF statements and UPDATE statements.

## Files

- Scenario1_LoanInterestDiscount.sql
- Scenario2_VIPPromotion.sql
- Scenario3_LoanReminder.sql

## Concepts Used

- Cursor
- FOR LOOP
- IF Statement
- UPDATE
- COMMIT
- DBMS_OUTPUT

## Scenarios

### Scenario 1

Apply 1% discount to loan interest for customers above 60 years.

### Scenario 2

Promote customers to VIP if balance exceeds $10,000.

### Scenario 3

Print reminders for loans due within the next 30 days.
