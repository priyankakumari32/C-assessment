# Stored Procedures

## Objective

Implement PL/SQL Stored Procedures for banking operations.

## Files

- Scenario1_ProcessMonthlyInterest.sql
- Scenario2_UpdateEmployeeBonus.sql
- Scenario3_TransferFunds.sql

## Concepts Used

- Stored Procedures
- Parameters
- Variables
- UPDATE Statement
- IF Statement
- COMMIT
- DBMS_OUTPUT

## Scenarios

### Scenario 1
Apply 1% monthly interest to all savings accounts.

### Scenario 2
Update employee salary by adding a bonus percentage for a specific department.

### Scenario 3
Transfer funds between two accounts after checking sufficient balance.
