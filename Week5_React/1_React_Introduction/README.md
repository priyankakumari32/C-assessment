# React Introduction

## Objective

Create the first React application using Create React App.

## Prerequisites

- Node.js
- npm
- Visual Studio Code

## Steps

1. Install Node.js and npm.
2. Create React app using Create React App.
3. Open the project in VS Code.
4. Modify App.js.
5. Run the application using npm start.

## Output

The application displays:

Welcome to the first session of React

The output screenshot is available in the output folder.
